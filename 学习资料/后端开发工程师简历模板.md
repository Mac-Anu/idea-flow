## 项目经验

### IdeaFlow - 智能全栈知识管理引擎 (独立开发)

`Next.js` `TypeScript` `Hono RPC` `Drizzle ORM` `TailwindCSS` `LangGraph` `Zustand`

- **项目描述**：
    独立从零架构并研发的一款媲美 Notion 体验的沉浸式知识管理与写作平台。采用现代化的 BFF (Backend For Frontend) 架构，结合 AI 智能体技术，极大提升了内容创作的效率与交互体验。
- **核心开发工作与亮点**：
  - **端到端类型安全架构**：抛弃传统的 REST API，采用轻量级 `Hono RPC` 结合 `Zod` 和 `Drizzle ORM`，实现了从数据库、后端校验到前端调用 100% 的类型推导，彻底消除 API 数据结构不同步的隐患。
  - **极简沉浸式富文本交互**：基于 Headless 框架 `Tiptap` 深度定制了富文本编辑器。通过抽取 Custom Hooks 分离 UI 与逻辑层，结合 `use-debounce` 实现了高效的防抖静默自动保存机制；并利用 `IntersectionObserver` 实现了右侧目录的精确滚动跟随。
  - **动态响应式 UI 与主题系统**：摒弃臃肿的组件库，使用 `TailwindCSS` 并自建 CSS Semantic Variables (语义化变量)，实现了完美的深浅色模式无缝热切换和细腻的微交互动画。
  - **AI 智能体工作流落地 (Self-Reflection Agent)**：拒绝简单的单轮对话 API 调用，深入底层手写 `LangGraph` 状态图，构建了包含生成器 (Generator) 和审查器 (Critic) 的对抗式循环状态机。实现了代码级的 AI 质量检测与多轮强制修正循环，大幅提升 AI 辅助写作的逻辑严密性。
