# Tailwind 视角的 CSS 核心心法速查

前端有成百上千个 CSS 属性，而在 Tailwind 的世界里，你**只需掌握以下这三个核心心法**，就能搞定 95% 的页面布局。其他花里胡哨的阴影、渐变、动画效果，需要用的时候直接问 AI 即可。

---

## 一、盒子模型 (Box Model) —— 东西是怎么撑大的？

在网页的眼里，**一切皆是矩形盒子**（无论它看起来是不是圆角）。一个盒子由内到外由四个部分组成。

- **`w-` / `h-` (Content / 实际内容)**：文字或图片本身的宽高。(如：`w-full` 满宽, `w-16` 宽 64px, `h-screen` 填满全屏高度)
- **`p-` (Padding / 内边距)**：内容到盒子边缘的缓冲距离。**它是填充在盒子内部的**。如果盒子有红色背景，padding 区域也是红色。(如：`p-4` 四周加 padding，`px-4` 左右加，`pt-4` 上面加)
- **`border-` (Border / 边框)**：盒子的分界线。
- **`m-` (Margin / 外边距)**：盒子排斥其他盒子的距离。**它是盒子外面的空白**。(如：`m-4` 让它离别人远点，`mx-auto` 经典的左右自动边距 = 水平居中)

> **💡 实战技巧**：不知道是用 `p` 还是 `m` 时？
> - 想让边框/背景色变大 ➔ 用 `p` (把里面撑肥)
> - 想把旁边的元素推远一点 ➔ 用 `m` (和别人保持距离)

## 二、Flex 布局 (Flexbox) —— 终极排版杀器 (🌟 最重要)

传统 HTML 是像文档一样从上往下堆叠的，怎么让它们并排？怎么让它们上下左右居中对齐？全靠 Flex 布局。只要在父容器上写一个 `flex`，里面的子元素就全都变成了听话的弹性方块。

### 1. 排列方向 (Direction)
写了 `flex` 后，子元素默认是从左到右排成一行的。
- **`flex-row` (默认)**：自左向右排（水平）。
- **`flex-col`**：自上向下排（垂直，像叠罗汉一样）。

### 2. 水平对齐与垂直对齐 (⭐ 面试必考级别)
一旦指定了方向，就会产生两个轴：**主轴** (沿着你们排队的轴)，和 **侧轴** (和排队方向垂直的轴)。

假设方向是默认的 `flex-row` (从左到右)：
- **`justify-` 开头（控制主轴：左右对齐）**
  - `justify-start`: 全靠左 (默认)
  - `justify-center`: 全部居中聚集！
  - `justify-between`: 两端对齐（左边头像靠最左，右边按钮靠最右，全网最常用！）
- **`items-` 开头（控制侧轴：上下对齐）**
  - `items-start`: 全靠天上对齐
  - `items-center`: 上下绝对居中对待（告别旧时代繁琐的居中代码！）

> **🏆 黄金法宝（无敌居中术）**：遇到任何东西你想让它在中间，直接在它外壳写：`flex justify-center items-center`。

### 3. `gap-` (元素间隙)
这是 Flex 提供的神器。如果你有 3 个按钮想并排，中间留点空隙，千万别去给每个按钮写外边距 (`margin`)！
直接在包含这三个按钮的父级上写 `flex gap-4`，自动在按钮之间插入标准空隙。

## 三、定位 (Position) —— 脱离地球引力

大部分元素都在大地上排队（这叫文档文档流），但有时你需要一些“悬停”在上面的元素（比如回到顶部按钮、弹窗覆盖层）。

- **`relative` (相对定位 - 锚点)**：它仍然在大地上老老实实排队。但它向天大吼一声：“我是**坐标原点**！”
- **`absolute` (绝对定位 - 卫星)**：**脱离大地！完全不占空间。** 只要写了 `absolute`，它就会像鬼魂一样漂浮起来。并且它会立刻往上寻找最近的一个声明了 `relative` 的祖先元素，以此为原点使用 `top-0`、`right-0` 等进行定位。
  > 实战：图片右上角加个红点。图片容器设为 `relative`，红点设为 `absolute top-0 right-0`。
- **`fixed` (固定定位 - 太阳)**：更霸道！它不仅脱离大地，连原来的爹也不认了，它以**浏览器的窗口**为参照物。
  > 实战：网页最上面的固定导航栏 (`fixed top-0 w-full`)。不管网页怎么往下滚，它永远钉死在屏幕顶端。
- **`sticky` (粘性定位 - 变色龙)**：平常像普通元素一样排队，当你往下滚动页面，它快要滚出屏幕时，突然像有胶水一样“啪”地黏在屏幕顶部不走了。（例如 Notion 文章大纲那一栏）。

---

## 🚀 总结
在实战的时候，如果大脑空白不知道怎么布局，按照这个思路闭眼问自己：
1. **这里要并排吗？要居中对齐吗？** ➔ 立刻敲下 `flex`！
2. **两边太挤了？** ➔ 加个 `p-4` 撑起来！
3. **要有悬浮红点/固定遮罩层吗？** ➔ 父传 `relative`，子设 `absolute`！
